#Condições

```
if (condicao) {
	//bloco
	true
} else {
	//bloco
	false
}
```

##Tipos de Condição

###Condição Simples

```
if (codicao) {
	true
}
```

###Condição Múltipla

Possibilidades de outros valores

```
switch (expressão) {
	case 1:

		break
	case 2:

		break
	case 3:

		break
	defalut:

		break
}
```

- O uso do break é obrigatório.
